# Backlog BI

Summarize the backlog task status into a chart

## how to use
1. install npm or yarn
2. type `npm install`
3. type `npm run dev`
4. open `http://localhost:3000`

## how to build
 
1. type `npm generate`

## used library

| name | url |
----|---- 
| element-ui | https://element.eleme.io/#/en-US |
| chart.js | https://www.chartjs.org/ |
|vue-chart.js|https://vue-chartjs.org/|
